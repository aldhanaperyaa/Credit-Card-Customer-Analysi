Credit Card Customer Analysis – Power BI

This project focuses on analyzing customer spending and repayment behavior using Power BI. The dashboard was built to understand transaction patterns, customer segments, revenue contribution, and repayment trends through interactive visualizations and KPI tracking.

The project contains multiple dashboard pages covering overall business performance, customer analysis, spending trends, and repayment behavior. Different charts, filters, and slicers were used to make the dashboard interactive and easy to explore.
